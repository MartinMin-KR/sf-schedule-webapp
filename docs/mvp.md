# Silicon Valley Journal MVP

## 목표

- 학생이 day/session 템플릿만으로 1분 안에 기록을 남기고,
- 집단 기록은 raw feed가 아니라 `Session Snapshot`, `Question Board`, `Day Recap`으로 다시 소비되게 한다.

## 핵심 결정

- 모바일 우선
- `홈 / 기록 / 탐색 / 내 기록` 4탭
- 일정 미정 상태를 기본 전제로 설계
- 개인 기록과 전체 기록을 시각적으로 분리
- 저장 상태와 공개 범위를 명시

## 이번 구현 범위

- day/session 템플릿 데이터
- 개인 기록 작성 폼
- 로컬 저장 기반 내 기록 아카이브
- 집단 요약 카드
- 질문 보드
- 대표 원문 카드

## 제외 범위

- 실시간 다중 사용자 동기화
- 로그인/권한 시스템
- 관리자 moderation 화면
- AI 자동 요약
- 사진 업로드
